# Controls Evidence Review Skill

Use this skill when the user asks whether SOX/control evidence is complete, sufficient, timely, or review-ready.

Workflow:
- Identify control objective, control frequency, preparer, reviewer, and evidence period.
- Check whether evidence demonstrates operation of the control.
- Look for preparer/reviewer signoff, dates, precision, exception handling, and support quality.
- Distinguish control evidence gaps from underlying accounting errors.
- Return open items that control owners can remediate.

Required inputs:
- Control description or control ID.
- Evidence file or close package support.
- Period and control frequency.

Output style:
- Lead with evidence sufficiency.
- List missing attributes and remediation actions.
- Avoid giving audit assurance.

Common failure modes:
- Treating a screenshot as sufficient without date, reviewer, or precision evidence.
- Missing population/sample support.
- Confusing review evidence with performance evidence.
