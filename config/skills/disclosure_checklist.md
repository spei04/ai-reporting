# Disclosure Checklist Skill

Use this skill when the user asks what disclosures are required, what support should be prepared, or how to plan a reporting checklist.

Workflow:
- Identify the accounting topic, filing context, period, and transaction/fact pattern.
- Retrieve rule context for the topic.
- Convert rule requirements into a practical reporting checklist.
- Separate required disclosures, support needed, judgments, and open questions.

Required inputs:
- Topic or transaction.
- Filing/reporting context if the checklist depends on SEC vs internal reporting.

Output style:
- Keep checklist items short and action-oriented.
- Include likely source documents needed for each item.
- Flag where legal/accounting policy judgment is needed.

Common failure modes:
- Returning generic disclosure theory instead of preparer actions.
- Missing source support requirements.
- Overstating requirements when facts are incomplete.
