# AI Reporting Feature Map

This folder records the current web app and backend features by product area.

- `web-interface.md` - chat UI, sidebar, file library, and visual behavior.
- `chat-and-model-provider.md` - GPT-backed chat flow and fallback behavior.
- `reporting-skills.md` - backend reporting skill router and skill purposes.
- `scf-generation.md` - cash-flow generation workflow and artifacts.
- `knowledge-base.md` - shared ASC/SEC and private session knowledge.
- `session-files-and-artifacts.md` - uploads, outputs, downloads, and evidence artifacts.
- `slack-integration.md` - first Slack interface and planned usage.
- `deployment.md` - Vercel preview deployment path and runtime behavior.
