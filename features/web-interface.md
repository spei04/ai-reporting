# Web Interface

## Purpose

Provide a simple ChatGPT-style reporting assistant interface for reporting teams.

## Current Features

- Minimal chat homepage with greeting and centered composer.
- Light and dark mode.
- Marble-inspired visual texture.
- Suggested prompts for SCF drafting, SEC filing, source trace, and ASC rule questions.
- File attachment support from the composer.
- Left sidebar with:
  - Chat view.
  - Files and rules library.
  - Session upload panel.
- Sliding files/rules panel with tabs for:
  - Session uploads.
  - Generated outputs.
  - Shared ASC/SEC rules.
- Generated artifact drawer for downloadable workbooks and evidence files.
- Source citation cards can open a source details drawer without creating downloadable outputs for ordinary Q&A.
- SCF source trace preview rows open an evidence drawer with output value, formula, source workbook cells, rule reference, and review status.
- Disclosure completeness responses render checklist items in a table with status, disclosure item, rule, and note columns.

## Notes

The UI is intentionally compact and work-focused. It avoids a landing page and opens directly into the reporting assistant workflow.
